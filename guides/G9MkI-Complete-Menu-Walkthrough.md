# Lumix G9 Mark I — Complete Menu Walkthrough

This document lists EVERY menu item for the Panasonic Lumix G9 Mark I (DC-G9, DC-G9L).
We will go through each one so Nelson can confirm his chosen settings.

**Legend:**
- **Default** = Factory default value
- **Nelson** = Nelson's chosen value (to be filled in)
- Items marked with ⚙️ are saved to Custom Modes (C1, C2, C3-1 through C3-3)
- Items marked with 🔧 are global (NOT saved to Custom Modes)
- The G9 MkI uses [Rec] and [Motion Picture] menus (not [Photo] and [Video])
- Q.Menu supports 12 slots
- Custom Modes: C1, C2, C3 (with C3 having sub-positions C3-1 through C3-3)

## Key Differences: G9 MkI vs G9 II

| Feature | G9 MkI | G9 II |
|---------|--------|-------|
| Photo Menu | [Rec] | [Photo] |
| Video Menu | [Motion Picture] | [Video] |
| Custom Mode Slots | C1, C2, C3 (with C3-1 to C3-3 sub) | C1, C2, C3-1 to C3-10 |
| Q.Menu Slots | 12 items | 12 items (expandable in MODE2) |
| Advanced Features | 6K/4K Photo, Post Focus, HDR, Multi Exp, High Resolution Mode, Status LCD | V-Log, S&Q mode, Time Code, Subject Detection (Animal/Human/Car/etc) |
| No Animal/Human Detection | Subject detection NOT available | Full subject detection suite |
| No V-Log | Standard gamma curves only | V-Log support |
| No S&Q | Basic high-speed video | Full S&Q (Slow & Quick) |
| No Time Code | No time code support | Full time code support |
| AF Modes | Standard + Continuous | Standard + Continuous + Advanced tracking |

## NOTES FOR NELSON

**Items marked ⚙️ (saved to Custom Modes):** These settings are stored separately for each Custom Mode (C1, C2, C3-1/2/3), meaning you can have different configurations for different shooting styles.

**Items marked 🔧 (global):** These settings apply globally across all modes and cannot be customized per Custom Mode.

**Q.Menu:** Assign your 12 most-used settings to Q.Menu for one-button access during shooting (accessed by pressing Q button).

**Custom Mode Notes:**
- C1 and C2 are regular custom modes
- C3 has up to 3 sub-positions (C3-1, C3-2, C3-3) accessible via the mode dial
- You can save/load complete camera configurations to the [SETUP] > [Custom Mode Mem.] function

**Default Values:** These are factory defaults. Please review each setting and fill in your preferred values in the "Nelson" column. This will serve as your reference for future camera configuration changes.
