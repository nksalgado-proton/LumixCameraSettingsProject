# Lumix G9 II — Complete Menu Walkthrough

This document lists EVERY menu item for the Panasonic Lumix G9 II (DC-G9M2, fw 2.5).
We will go through each one so Nelson can confirm his chosen setting.

**Legend:**
- **Default** = Factory default value
- **Nelson** = Nelson's chosen value (to be filled in)
- Items marked with ⚙️ are saved to Custom Modes
- Items marked with 🔧 are global (NOT saved to Custom Modes)
- Items marked with 🆕 were added in firmware 2.3 or 2.5
